# Viram Tech - Frontend Website

This is the **frontend-only** React + TypeScript website for Viram Tech.

## Tech Stack
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Shadcn/UI

## Features
- Responsive, modern single-page site
- Contact form using mailto
- Static hosting ready

## Getting Started

```bash
npm install
npm run dev
